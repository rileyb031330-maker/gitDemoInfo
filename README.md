# Git Notes

## Common Commands

```bash
git add <file>
git add -A
git commit -m '<msg>'
git status

git branch
git checout -b <branchName>
git merge <branch>

git pull origin <branchName>
git push origin <branchName>

git log                   # 'q' to quit
git tag -a '<v1.0.0>' -m '<release msg>'

git reset --hard <id or tag or HEAD>
```
